Patient Demographics Query Service
==================================

The Patient Demographics Query Service queries external Patient Demographics Suppliers with QBP message to request
information about patients whose demographic data match data provided in the query message.
The Patient Demographics Query may be initiated using RESTful services or patient verification scheduler of |product|.

The functionality provided by the Patient Demographics Query Service of |product| is compliant with the requirements for
`IHE ITI-21 Transaction: Patient Demographics Query <https://www.ihe.net/uploadedFiles/Documents/ITI/IHE_ITI_TF_Vol2a.pdf#page=154>`_.
